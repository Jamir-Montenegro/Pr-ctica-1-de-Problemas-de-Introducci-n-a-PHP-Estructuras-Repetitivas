#  Ejercicios Básicos de PHP

Este repositorio contiene ejemplos prácticos y sencillos de programación en PHP. El objetivo es demostrar el uso de **operadores matemáticos**, **estructuras de control** y **arreglos** 

##  Temas Cubiertos

1.  **Operadores Matemáticos:**
    *   Suma (`+`), Resta (`-`), Multiplicación (`*`)
    *   División (`/`), Módulo (`%`), Exponenciación (`**`)
2.  **Orden de Precedencia:** Ejemplos de cómo PHP resuelve operaciones combinadas (con y sin paréntesis).
3.  **Estructuras de Control:** Uso de bucles `for`, `foreach` y condicionales `if`.
4.  **Arreglos (Arrays):** Recorrido, manipulación y visualización de datos.

##  Cómo ejecutar el proyecto

Estos ejercicios requieren un servidor local con soporte PHP (como XAMPP, Laragon o WAMP).

1.  Descarga o clona este repositorio.
2.  Copia los archivos `.php` en tu carpeta de servidor (por ejemplo: `htdocs` si usas XAMPP).
3.  Abre tu navegador web.
4.  Visita la dirección local (ej: `http://localhost/proyecto-php/index.php`).
5.  Verás los resultados de cada operación directamente en la pantalla.
